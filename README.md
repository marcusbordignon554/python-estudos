Este repositório reúne códigos desenvolvidos durante meus estudos em Python, abrangendo desde os fundamentos da programação até conceitos mais avançados de manipulação de dados, computação científica e desenvolvimento de aplicações.

O objetivo é documentar minha evolução na linguagem e reunir exemplos práticos dos conceitos estudados ao longo da disciplina.
